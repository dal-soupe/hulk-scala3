# Hulk scala 3

## Description

This is an updated version of [Maxime Morge's Hulk project](https://github.com/maximemorge/Hulk), adapted to be compatible with scala 3.

## Libraries needed

For compatibility between scala 3 and akka actors, librairies used were scala 3.6.4 anbd akka-actors 2.8.8

## sbt project compiled with Scala 3

### Usage

This is a normal sbt project. You can compile code with `sbt compile`, run it with `sbt run`.

For more information on the sbt-dotty plugin, see the
[scala3-example-project](https://github.com/scala/scala3-example-project/blob/main/README.md).
